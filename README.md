# Easy Habit Pro

A powerful and intuitive habit tracking application built with React and Supabase.

## Features

- Track daily, weekly, and custom frequency habits
- Visualize progress with streaks and charts
- Secure authentication with Supabase
- Responsive and modern UI

## Getting Started

### Prerequisites

- Node.js 18+
- A Supabase account

### Setup

1.  **Clone the repository** (if you haven't already):
    ```bash
    git clone <repository-url>
    cd habit-tracker4
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Environment Setup:**
    - Create a `.env` file in the root directory.
    - Add your Supabase credentials (see [SETUP.md](./SETUP.md) for details).

### Running the App

Start the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
